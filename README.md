# Flipper fools site

A site to capture the fools of the Flipper Zero discord in the wild
This site exists purely as entertainment and is not meant to be offensive. 

Site built with Hugo using the Classless theme. 
Be sure to use `git clone --recursive` as this uses submodules to provide the themes. 

`hugo server` to preview changes

`hugo new content content/posts/your-post-name/index.md` for a new post
